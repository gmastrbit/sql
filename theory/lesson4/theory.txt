Оператор LIKE в MySQL — вибірка діапазону значень

\c - очистити введення в командний рядок

IN - оператор, який дозволяє включати певний діапазон значень.

BETWEEN - оператор, який допомагає вибирати дані з певного діапазону.

NULL
Це не тип і не значення.
Це відсутність значення.

NOT NULL - данне поле обов'язково повинне бути заповнене.

LIKE - оператор для роботи з рядками.

Рекомендується використовувати для типів полів: char i varchar

Ефективно працює з невеликими БД.
Він не використовує в своїй роботі індекси.

% - невідома частина, ставить спереду або ззаду, або з обох сторін, будь-яка кількість будь-яких символів

_ - для невідомого символа

\_ - екранування

\% - екранування

LIKE шукає по всій БД.
